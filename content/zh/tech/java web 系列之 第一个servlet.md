
# 1.配置tomcat
教程如下

推荐的是设置Path 的时候不要用相对的路径,就是
```
%CATALINA_HOME%\bin\
```
最好用绝对路径,类似下面的
```
tomcat8.0\bin\
```
部分windows 版本只支持绝对路径, 以防万一,建议用以上的写法
